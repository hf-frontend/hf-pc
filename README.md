# hf-pc
>  用npm创建项目
>  全局安装React-cli
	npm install -g create-react-app
>  创建项目	
	create-react-app hf-pc
>  暴露项目的配置，可以自由配置项目所需的依赖，零配置即可开发。
	npm run eject
>  运行项目
	npm start
>  打包项目
	npm run build
>  安装google浏览器扩展
	https://chrome.google.com/webstore/category/extensions?hl=zh-CN
	搜索react developer tools并添加

# 目录结构
